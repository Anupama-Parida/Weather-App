# Weather App
A simple and clean iOS Weather application built using **Swift** and **SwiftUI**.  
The app fetches real-time weather data from the API, stores it locally using **Core Data**, and provides a smooth user experience.

### Features

- Search weather by city name
- Fetch real-time weather data using REST API
- Offline data persistence using Core Data
- Clean and modern SwiftUI UI
- Custom launch screen

### Tech Stack

- **Language:** Swift  
- **Framework:** SwiftUI  
- **Architecture:** MVVM  
- **Networking:** URLSession  
- **Local Storage:** Core Data  
- **API:** OpenWeatherMap API
